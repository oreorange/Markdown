# ストラテジ
## 1. 企業活動
```
用語・試験頻出
- CSR (Corporate Social Responsibility) 企業の社会的責任
- CIO (Chief Information Officer) 最高情報責任者 情報システムの統括　
- HRTech 人事関連業務においてAIやIoTといったITを活用する手法のこと
- パレート図　ABC分析　ヒストグラム　レーダーチャート　「散布図」　回帰分析
- 期待値 値と確率を掛け算し、全ての組み合わせを足し算して求める
- 損益分岐点　計算問題
- 財務諸表　貸借対照表、損益計算書、キャッシュフロー計算書
- 資本金　株主が会社に出資した金額、貸借対照表の純資産に分類される
- 自己資本比率とは、自己資本 / 総資本 * 100
- ROE (Return On Equity) 自己資本利益率　株主から集めたお金でどれだけの利益を上げたのかを表す指標
- 事業とは、複数の業務のこと。業務とは手作業とシステムのこと。システムとはハードウェアとソフトウェアのこと。
- 事業部制組織は、意思決定権とともに利益責任も負う
- ステークホルダー　企業と利害関係にある人　従業員、株主、取引先、消費者、行政機関、地域社会　全てのステークホルダーに利益を
- ブレーンストリーミング　複数人で集まってアイデアを出し合う方法
- キャッシュフロー計算書　現金の出入りを「営業活動」「投資活動」「財務活動」の３つに分類して計算する
```

## 2. 法務
```
用語・試験頻出
- 知的財産権（著作権、産業財産権、その他の権利）の３つに大きく分類される。
- 著作権　取扱説明書、プログラム、写真も保護される。プログラム言語、アルゴリズム、プロトコルは保護されていない。
- ビジネスモデル特許、例　1clickで今すぐ買う。
- サイバーセキュリティー基本法　サイバー攻撃から国を守るための基本理念を定めた法律、法律の対象者は国全体。
- セキュリティーホールからのアクセス（脆弱性をついたアクセス）も不正アクセス禁止法の対象になる。
- 契約形態によって、著作権の帰属先が変わる。
- 営業秘密とは（秘密管理性、有用性、非公知性）の３つの条件を満たす情報。
- 個人情報取扱事業者　届出は必要なし、1つでも個人情報を扱うと、個人情報取扱事業者になる。
- 該当しない組織（国の機関、地方公共団体、独立行政法人、地方独立行政法人）、国立大学法人や国税庁も該当しない。
- 情報公開法　行政機関が持つ情報の公開を、誰でも（法人や外国人でも）求めることができる。開示請求できるのは行政文書。
- ISO 9000シリーズ（品質）、14000（環境）、27000（情報セキュリティ）、番号が小さいほど古い規格である。
```

## 3. 経営戦略マネジメント
```
用語・試験頻出
- SWOT分析 経営環境を、Strengths(強み),Weaknesses(弱み),Opportunities(機会),Threats(脅威) の4つに分類する手法
- 内部環境によるものか外部環境によるものかで、分類する。
- PRM(Product Portfolio Management)　市場成長率や市場占有率から投資する対象を見極める。
- マーケティングミックス　4P (Product(製品),Price(価格),Place(流通),Promotion(販売促進))
- 4C (Customer Value(顧客にとっての価値), Cost(顧客の負担), Convenience(利便性), Communication)
- RFM分析　Recency(最終購入日), Frequency(購入頻度), Monetary(累計購買金額)。優良顧客を見つけるための分析手法
- UX 商品やサービスなどを実際に利用することによって得られる体験。
- BSC バランススコアカード　財務、顧客、業務プロセス、学習と成長の4つの視点から業績評価を行う手法。
- CSF (Critical Success Factors) 重要成功要因、戦略目標を達成するために必要となる具体的な要因。
- KPI (Key Performance Indicator) 重要業績評価指標、戦略目標の達成状況を表す指標。
- 一元管理の4用語 (ERP 社内の経営資源, CRM 顧客情報, SFA 営業ノウハウ, SCM 社外も含めた製品情報）
```
